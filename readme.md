# 🎙️ AI Voice Agent

A simple **voice-controlled web app** that performs actions like playing songs on YouTube using voice commands.

---

## 🚀 Features

- 🎤 Voice input (Speech Recognition API)
- ⚡ Play songs on YouTube
- 🔊 Voice response (Text-to-Speech)
- 🧾 Live transcript display
- ⏱ Auto-stop mic after 60 seconds of silence

---

## 🏗️ Tech Stack

- Next.js (TypeScript + Tailwind)
- Node.js + Express
- Puppeteer

---

## 📂 Setup

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🎯 Usage

Click mic and say:

```
Play Arijit Singh
Play lo-fi music
```
